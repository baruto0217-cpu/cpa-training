# CPA Training — 愛知県救急活動訓練シミュレーション記録

愛知県第18版に対応した、救急活動訓練シミュレーション用の記録・評価アプリです。  
iPadのホーム画面に追加してPWAとして使用できます。

---

## 🌐 使い方（GitHub Pages）

👉 **[アプリを開く](https://あなたのGitHubユーザー名.github.io/cpa-training/)**

iPadのSafariで上記URLを開いてください。

---

## 📲 iPadへのインストール手順

1. iPadの **Safari** でアプリのURLを開く
2. 画面下の **共有ボタン（□↑）** をタップ
3. **「ホーム画面に追加」** を選択
4. 名前を確認して **「追加」** をタップ
5. ホーム画面のアイコンからアプリとして起動 🎉

> ホーム画面から起動するとアドレスバーが消え、全画面アプリとして動作します。

---

## ✨ 機能一覧

| 機能 | 説明 |
|---|---|
| 📋 想定票入力 | 傷病者情報・ピリオド経過の記録 |
| ✅ 評価チェックリスト | 活動項目のO/X評価（編集可能） |
| ⏱️ 活動タイマー | 経過時間の計測・メモ追記 |
| 🎵 メトロノーム | 胸骨圧迫テンポ（100〜120 BPM） |
| ✏️ 手書きメモ | タッチ・Apple Pencil対応キャンバス |
| 📊 結果レポート | ×判定一覧・達成率表示 |
| 💾 記録保存 | ローカルストレージへの複数記録保存 |
| 📁 ファイル書出/読込 | JSONファイルへのエクスポート/インポート |
| 📄 PDF出力 | A4横向きPDFレポートの生成 |

---

## 🛠️ 技術仕様

- **フロントエンド**: 純粋なHTML / CSS / JavaScript（フレームワーク不使用）
- **データ保存**: localStorage（サーバー不要・完全オフライン動作）
- **PWA対応**: `apple-mobile-web-app-capable` によるホーム画面追加対応
- **PDF生成**: html2canvas + jsPDF（CDN読み込み）
- **外部依存**: Google Fonts（Noto Sans JP / JetBrains Mono）のみ

## 🚀 GitHub Pagesへの公開手順

1. このリポジトリを GitHub にプッシュ
2. リポジトリの **Settings** → **Pages** を開く
3. Source を **「Deploy from a branch」** に設定
4. Branch を **`main`**、フォルダを **`/ (root)`** に設定して **Save**
5. 数分後に `https://ユーザー名.github.io/リポジトリ名/` で公開完了

---

## ⚠️ 注意事項

- データはブラウザの **localStorage** に保存されます
- ブラウザのデータ消去を行うと保存記録も削除されます
- 大切な記録は **「書出」ボタン** でJSONファイルとしてバックアップしてください
- PDF出力にはインターネット接続が必要です（ライブラリをCDNから読み込むため）
